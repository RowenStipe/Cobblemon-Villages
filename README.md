This is a repo for the community modpack for Rei Sirena's community.

This modpack is designed around the idea of village building and exploration with Cobblemon as it's core.

## Contributing
Contributions are welcome. Check the issues or project board for things that need to be worked on.

## Development
This pack is developed with CurseForge as the primary platform. The files in `Source` are used to create the modpack. You can create a zip file of this folder for setting up a development environment in CurseForge.
